How to Use Gender Detection

1. Upload an Image

Click the "Upload Image" button or drag and drop an image containing one or more faces.

2. Face Detection

The Face++ API will automatically detect all faces in the image.

3. Gender Analysis

For each detected face, the system will determine gender and display the results with confidence scores.

4. View Results

Results will show each face with gender classification and additional attributes like age and emotion.

Note: This tool uses Face++ API for gender detection. Results are based on visual features and may not be 100% accurate in all cases.
